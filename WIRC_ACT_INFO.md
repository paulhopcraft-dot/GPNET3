## Act Structure (Known Sections)

Based on WorkSafe Claims Manual references, key sections include:

### Weekly Payments & Entitlements
- **Section 37** - Step-down payments after 13 weeks
- **Section 38** - Certificate of capacity requirements
- **Section 99** (AC Act ref) - Return to work obligations

### Return to Work & Rehabilitation
- **Section 41** - RTW plan requirements (10-week timeline)
- **Section 155-165** - RTW obligations

### Medical Services
- Sections covering medical treatment entitlements
- Certificate requirements
- Medical examinations

### Dispute Resolution
- Sections covering conciliation and dispute processes

### Regulations
- **WIRC Regulation 224** - 8-week file review requirement

## Key Compliance Points

### 1. Certificate Requirements (s38)
- Worker off work must have current certificate of capacity
- Certificate must specify capacity level (fit/partial/unfit)
- Employer can request certificate
- Payments can be suspended if certificate expires

### 2. Weekly Payments (s37)
- First 13 weeks: Higher payment rate
- After 13 weeks: Step-down if not returned to work with pre-injury employer
- Maximum weekly payments apply
- PIAWE (Pre-Injury Average Weekly Earnings) calculation

### 3. Return to Work (s41, s99)
- Employer must provide suitable duties where reasonable
- RTW plan required for serious injuries (within 10 weeks)
- Graduated return to work preferred
- Employer obligations to cooperate

### 4. File Reviews (WIRC Reg 224)
- Agent must review file every 8 weeks maximum
- Review must assess current work capacity
- Determine ongoing entitlements

### 5. Medical Services
- Worker entitled to reasonable medical treatment
- Employer/agent can request medical examinations
- Treatment must be related to work injury
